# 🛡️ Offensive Security Studies & Labs

Este repositório centraliza meus estudos práticos e documentações sobre segurança ofensiva, protocolos de rede e análise de vulnerabilidades. O objetivo principal é entender os vetores de ataque para construir defesas mais robustas e aplicar os princípios de **DevSecOps** no ciclo de vida do software.

## 📂 Estrutura dos Laboratórios

Os estudos estão divididos por protocolos e tipos de ataques, contendo o passo a passo completo da exploração.

## 🔒 Aviso Ético (Disclaimer)

Todo o conteúdo deste repositório foi desenvolvido em ambientes controlados e laboratórios de estudo (como máquinas virtuais e plataformas de treinamento).O objetivo é exclusivamente educacional e focado em segurança defensiva e mitigação de riscos.
